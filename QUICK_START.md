# 🚀 快速開始指南

## 立即運行應用程式

### 方法 1: 使用啟動腳本（最簡單）

```bash
cd mindful-mind-app
./START.sh
```

### 方法 2: 使用 npm 命令

```bash
cd mindful-mind-app
npm install  # 僅首次需要
npm run dev
```

## 訪問應用

打開瀏覽器訪問：**http://localhost:3000**

## 首次使用

1. 在首頁輸入你的名字
2. 點擊「立即開始」
3. 開始使用各項功能！

## 主要功能快速導覽

### 1️⃣ AI 心靈助手（/chat）
- 24/7 與 AI 對話
- 獲得情緒支持和建議
- 支持對話歷史保存

**快速提問範例：**
- "我感到焦慮，該怎麼辦？"
- "如何改善睡眠品質？"
- "幫我做呼吸練習"

### 2️⃣ 情緒追蹤（/mood）
- 記錄每日心情（1-5 分）
- 添加標籤（工作、家庭、健康等）
- 寫下感受和想法

### 3️⃣ 冥想與呼吸（/meditation）
- 選擇時長（3/5/10/15 分鐘）
- 跟隨呼吸引導
- 記錄冥想時間

### 4️⃣ 每日日記（/journal）
- 記錄生活點滴
- 使用靈感提示開始寫作
- 保存和回顧過往日記

### 5️⃣ 數據儀表板（/dashboard）
- 查看心情趨勢圖表
- 追蹤連續天數
- 檢視整體統計

### 6️⃣ 資源庫（/resources）
- 心理健康文章
- 引導音頻
- 緊急求助專線

## 技術要求

- Node.js 18 或更高版本
- 現代瀏覽器（Chrome、Firefox、Safari、Edge）
- 穩定的網絡連接（用於 AI 對話）

## API 配置

Groq API 密鑰需要配置在 `.env` 文件中：
```env
GROQ_API_KEY=your-groq-api-key-here
```

應用會自動從環境變數中讀取 API 密鑰。

## 常見問題

### Q: AI 對話無回應？
A: 確保：
1. 後端服務器正在運行（http://localhost:3001）
2. 網絡連接正常
3. Groq API 密鑰有效

### Q: 數據會丟失嗎？
A: 所有數據保存在瀏覽器的 LocalStorage 中，除非清除瀏覽器數據，否則不會丟失。

### Q: 可以在手機上使用嗎？
A: 可以！UI 完全響應式，但需要在同一網絡下訪問本地服務器。

### Q: 如何部署到線上？
A: 查看 `DEPLOYMENT.md` 獲取詳細部署指南。
